== Description ==

This plugin puts the ID of a post in the slug (also known as title or name).
External page of the plugin:
http://en.yarri.org/wordpress-plugin-id-in-slug/

== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/yarri-id-in-slug` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Yarri's "ID in slug" screen to configure the plugin
